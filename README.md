**IPL Dashboard - Power BI**

This project is an interactive IPL Dashboard created using Power BI to analyze IPL match data, team performances, player statistics, win trends, and season insights through dynamic and visually appealing dashboards.

**Features**

Team-wise performance analysis
Player statistics and comparisons
Match result insights
Season trends and win analysis
Interactive filters and slicers
Dynamic visualizations and KPIs

**Tools Used**

Power BI
Excel / CSV Dataset
DAX
Data Cleaning & Visualization

**Project Objective**

To transform IPL datasets into meaningful visual insights that help users understand team strategies, player performance, and match outcomes effectively.
